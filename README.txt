# myChat_JR
Мини-проект "Чат" 
Проект выполнен в ходе прохождения обучения на JavaRush.

Для запуска программы необходимо:
1. Запустить сервер(Server.java);
2. Указать номер порта через который будем работать;
3. Всё сервер запущен, что подтвердит надпись в консоле;
4.Теперь запускаем клиент, чтобы запустить графический клиент 
запускаем ClientGuiController.java,чтобы запустить консольную 
версию - запускаем Client.java(для теста можно запустить несколько 
клиентов);
5. При запуске клиентов указываем адрес сервера (localhost) и номер 
порта указанный в п.2;
6. Также можно запустить бота, после запуска он сообщит всем 
участникам чата, что он умеет и как этими умениями воспользоваться.
